# 愛媛・道後温泉 三泊四日 確定プラン
## 5月14日（水）〜 17日（土）｜夫婦ドライブ旅

**コンセプト:** ゆったりドライブ × 道後温泉外湯巡り × 文化体験  
※妻の体調を最優先。激しいアクティビティなし。休憩・トイレ多め。

---

## 旅の全体像

```
Day0（5/14 水）移動日
  羽田 18:00発 → 松山 19:30着
  レンタカー受け取り → 東横イン 松山大街道（格安・素泊まり）

Day1（5/15 木）
  ドライブ①：松山市内 → 砥部（20分・田舎道）
  砥部焼絵付け体験 → 大街道グルメ → 道後hakuro チェックイン
  夕方：道後温泉エリア散策 → 足湯 → ウラ道後（夜）

Day2（5/16 金）
  ドライブ②：道後 → 下灘駅（50分・海沿いの県道）
  下灘駅でのんびり → 内子・大洲 町歩き
  ドライブ③：下灘 → 内子・大洲（50分・山あいの国道）
  夜：道後温泉本館（短時間入浴） or 飛鳥乃湯泉（足湯プランで）

Day3（5/17 土）
  ドライブ④：しまなみ海道（今治 → 大三島）
  大山祇神社・宝物館 → 帰路ドライブ → 松山空港 16:00発
```

---

## ドライブについて（免許取り立ての方へ）

**愛媛の地方道は新しいドライバーに向いています。**

| ルート | 道の特徴 | 難易度 |
|--------|---------|--------|
| 松山 → 砥部（県道183号） | 空いた田舎道・直線多め | ★☆☆☆☆ |
| 道後 → 下灘（県道378号） | 海沿い・交通量少・景色最高 | ★★☆☆☆ |
| 下灘 → 内子（国道56号） | 山あいの国道・適度なカーブ | ★★☆☆☆ |
| しまなみ海道（今治 → 大三島） | 自動車専用道・走りやすい | ★★☆☆☆ |

**レンタカー選び:** コンパクトカー（ヤリス・ノート等）が最初は運転しやすい。  
**注意点:** 妻の体調優先で、1〜1.5時間に一度は必ず休憩・トイレ休憩を。

## 宿：道後hakuro（2泊）

大浴場なし・外湯巡り前提の宿。妊娠の可能性がある場合でも温度・時間を自分でコントロールしやすい。

- 全室レコードプレーヤー付き・フリードリンクラウンジ
- 「彩る朝食」ビュッフェ（6:20〜）
- 道後温泉駅から徒歩5分
- **¥20,000〜（2名・朝食付き）**
- 公式: [dogohakuro.jp](https://www.dogohakuro.jp/)

## 外湯について（妊娠の可能性がある場合）

> **重要:** 温泉入浴は出発前に産婦人科に確認することを強く推奨します。

| 施設 | 妊娠の可能性がある場合 | 備考 |
|------|----------------------|------|
| **道後温泉 足湯** | ◎ 比較的安全 | 温泉街に複数あり・無料〜100円 |
| **椿の湯・本館・飛鳥乃湯泉** | △ 短時間・ぬるめなら可 | 長湯・熱湯は避ける |
| **飛鳥乃湯泉 個室プラン** | △ 2人のペースで入れる | 時間・温度のコントロールがしやすい |

**3館周湯チケット ¥1,400/人**（2割引・有効期限2日間）  
購入は各館窓口で当日OK。

## この旅のテーマ

> 「走ることも、旅の一部にする」

砥部への田舎道、海沿いの夕やけこやけライン、しまなみ海道の橋の上。  
愛媛は走るだけで景色が変わる県。免許取り立ての最初のドライブ旅行として、ちょうどいいスケール。  
妻の体調を最優先に、無理せずゆったりと。
