# 愛媛・道後温泉 二泊三日 確定プラン（パターンB）
## 5月15日（木）〜 17日（土）｜夫婦旅

---

## 旅の全体像

```
Day1  羽田 6:50発 → 松山 8:20着
      松山空港でレンタカー受け取り
      松山城（裏ルートハイク）→ 砥部焼絵付け → 大街道グルメ
      → 道後温泉チェックイン → 飛鳥乃湯泉（個室） → ウラ道後（夜）

Day2  道後温泉 朝風呂（6:00〜）
      大洲・内子 町歩き → 下灘駅で夕日（〜19:00）
      → 道後温泉 又新殿見学 → 宇和島風鯛めし

Day3  朝風呂 → 瀬戸内カヤック/SUP（今治エリア）
      → 松山空港 16:00発 → 羽田 17:30着
```

**移動手段:** レンタカー（松山空港で受け取り・返却）

## 宿泊：道後温泉 おすすめ3択

### ★ 第1推奨「道後温泉 ふなや」
**創業390余年の老舗。道後温泉駅から徒歩3分。**

- **料金:** 2名1室 ¥37,400〜（1名¥18,700〜）、夕朝食付き
- **強み:** 1,500坪の日本庭園、大浴場（御影石「御影湯」＋古代檜「檜湯」、男女朝夕入替）
- **客室:** 和室・和洋室・庭園ビューなど多様
- **口コミ:** 夕食・朝食・接客・清潔感すべて高評価
- **公式 / 予約:** [dogo-funaya.co.jp](https://www.dogo-funaya.co.jp/) / じゃらん・楽天トラベルでも可

> 「変わった体験を求める夫婦」に合う理由：老舗の矜持と庭園の静けさ。夜の庭園散歩が非日常。

---

### ★ 第2推奨「ホテル道後やや」
**オリコン顧客満足度3年連続1位。カジュアルで飾らない新しい道後スタイル。**

- **料金:** 2名1室 ¥27,000〜（1名¥13,500〜）、朝食ビュッフェ付き
- **強み:**
  - **みかんジュース蛇口が飲み放題**（愛媛らしさ全開）
  - 浴衣を4種類から選べる
  - アメニティ充実、夜食の鯛めしあり
  - 大浴場なし → 外湯（飛鳥乃湯泉・本館・椿の湯）を巡る設計思想
- **公式 / 予約:** [yayahotel.jp](https://www.yayahotel.jp/) / じゃらん・トラベルコ

> 「温泉よりも外湯文化と街を楽しみたい」カップルにぴったり。泊まった宿で完結させない自由なスタイル。

---

### ★ 第3推奨「ホテル葛城 Spa Resort 道後」
**道後温泉本館から徒歩1分。コスパで選ぶならここ。**

- **料金:** 2名1室 ¥16,500〜（素泊まり）/ ¥26,730〜（夕朝食付き）
- **強み:** 本館まで徒歩1分の立地最強、大浴場あり（檜造り）、姉妹館「花ゆづき」の天望露天風呂も利用可
- **予約:** [じゃらん](https://www.jalan.net/yad301458/) / [楽天トラベル](https://hotel.travel.rakuten.co.jp/hotelinfo/plan/16718)

> 立地最優先で選ぶ場合の最有力。本館と飛鳥乃湯泉をはしごしやすい。

## この旅のテーマ

> 「ゆっくり動いて、深く感じる3日間」

道後温泉3000年の湯、江戸〜明治が残る城下町、夕焼けに染まる無人駅、瀬戸内の海を自分の腕で漕ぐ。  
詰め込まず、1つ1つの場所に時間を使う旅。それが「少し変わった体験」になる。
