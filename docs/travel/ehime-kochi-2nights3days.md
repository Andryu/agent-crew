# 愛媛・道後温泉 三泊四日 確定プラン
## 5月14日（水）〜 17日（土）｜夫婦ドライブ旅

**コンセプト:** ゆったりドライブ × 道後温泉外湯巡り × 少し香川も  
※妻の体調を最優先。激しいアクティビティなし。休憩・トイレ多め。

---

## 旅の全体像

```
Day0（5/14 水）移動日
  成田 18:00発（Jetstar） → 松山 19:35着
  レンタカー受け取り → 東横イン 松山大街道（格安・素泊まり）

Day1（5/15 木）
  松山市内グルメ → 坊っちゃん列車 → 道後温泉エリア散策
  道後hakuro チェックイン → 夕方足湯 → 椿の湯 → ウラ道後（夜）

Day2（5/16 金）
  ドライブ①：道後 → 下灘駅（50分・海沿いの夕やけこやけライン）
  下灘でのんびり → 内子 ランチ＆町歩き
  ドライブ②：内子 → 父母ヶ浜（香川・三豊市、1時間30分）
  父母ヶ浜で夕日のリフレクション撮影 → 道後帰着
  夜：飛鳥乃湯泉 個室プラン

Day3（5/17 土）
  ドライブ③：しまなみ海道（今治 → 大三島）
  大山祇神社・宝物館 → 帰路ドライブ → 松山空港 16:30発（Jetstar）→ 成田着
```

## ドライブについて（免許取り立ての方へ）

**愛媛〜香川の地方道は新しいドライバーに向いています。**

| ルート | 道の特徴 | 難易度 |
|--------|---------|--------|
| 道後 → 下灘（県道378号） | 海沿い・交通量少・景色最高 | ★★☆☆☆ |
| 下灘 → 内子（国道56号） | 山あいの国道・適度なカーブ | ★★☆☆☆ |
| 内子 → 父母ヶ浜（高松道/国道11号） | 高速あり・走りやすい | ★★☆☆☆ |
| しまなみ海道（今治 → 大三島） | 自動車専用道・走りやすい | ★★☆☆☆ |

**レンタカー選び:** コンパクトカー（ヤリス・ノート等）が最初は運転しやすい。  
**注意点:** 妻の体調優先で、1〜1.5時間に一度は必ず休憩・トイレ休憩を。

> **父母ヶ浜（ちちぶがはま）について**  
> 香川県三豊市の浜。干潮時に浜に水が薄く張り、夕日を映す「天空の鏡」現象が起きる。  
> 入場無料、駐車場あり（混雑時¥500程度）。撮影スポットとして全国から人が訪れる。  
> 内子からは松山道経由で1時間30分。愛媛に帰るついでに香川を通れる好立地。

> **「夕やけこやけライン」ドライブ補足**  
> 道後→下灘の県道378号沿いは「夕やけこやけライン」と呼ばれる絶景ルート。  
> 海岸ぎりぎりを走る区間があり、ドライブ自体が体験になる。

## この旅のテーマ

> 「走ることも、旅の一部にする」

海沿いの夕やけこやけライン、しまなみ海道の橋の上、そして香川の父母ヶ浜まで足を伸ばす。  
愛媛は走るだけで景色が変わる県。そしてちょっとだけ香川まで。  
免許取り立ての最初のドライブ旅行として、ちょうどいいスケール。  
妻の体調を最優先に、無理せずゆったりと。
