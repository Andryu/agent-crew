# 愛媛・高知 三泊四日 確定プラン
## 5月14日（水）〜 17日（土）｜夫婦ドライブ旅

**コンセプト:** ゆったりドライブ × 道後温泉外湯巡り × 高知グルメ（ひろめ市場）  
※妻の体調を最優先。激しいアクティビティなし。休憩・トイレ多め。

---

## 旅の全体像

```
Day0（5/14 水）移動日
  16:00 成田第3ターミナル集合
  18:00 成田発（Jetstar）→ 19:35 松山着
  レンタカー受け取り → 松山市内 格安ホテル

Day1（5/15 木）愛媛ドライブ × 道後温泉
  8:30 大街道グルメ → 下灘駅（夕やけこやけライン）→ 内子ランチ
  14:30 道後hakuro チェックイン
  夕方：足湯 → 飛鳥乃湯泉 個室プラン → ウラ道後

Day2（5/16 金）高知へ
  6:30 道後温泉本館 朝風呂 → 8:00 hakuro朝食 → チェックアウト
  9:15 松山発 → 11:30 高知着（高速2h10m）
  ひろめ市場（昼・夜）→ 高知城 → 帯屋町 → 高知泊

Day3（5/17 土）高知グルメ締め → 松山空港
  7:00 喫茶店モーニング → 10:00 ひろめ市場（朝カツオたたき）
  12:00 高知発 → 14:10 松山空港着
  16:30 Jetstar 成田行き
```

## この旅のテーマ

> 「走ることも、食べることも、旅の一部にする」

海沿いの夕やけこやけライン、四国山地を越えて辿り着く高知。  
ひろめ市場で昼から地酒を傾けるのが高知スタイル。  
免許取り立てでの四国横断。妻の体調を最優先に、無理せずゆったりと。
