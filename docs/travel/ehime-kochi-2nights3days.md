# 愛媛・高知 三泊四日 確定プラン
## 5月14日（水）〜 17日（土）｜夫婦ドライブ旅

**コンセプト:** ゆったりドライブ × 道後温泉外湯巡り × 高知グルメ（ひろめ市場）  
※妻の体調を最優先。激しいアクティビティなし。休憩・トイレ多め。

---

## 旅の全体像

```
Day0（5/14 水）移動日
  成田 18:00発（Jetstar） → 松山 19:35着
  レンタカー受け取り → 松山市内 格安ホテル

Day1（5/15 木）
  朝：大街道で軽くグルメ → ドライブ①下灘駅（夕やけこやけライン）→ 内子ランチ
  道後hakuro チェックイン → 足湯 → 飛鳥乃湯泉 個室プラン → ウラ道後

Day2（5/16 金）
  朝風呂（道後温泉本館）+ 朝食 → チェックアウト
  ドライブ②道後→高知（松山道・高知道、2時間10分）
  ひろめ市場（カツオたたき・安兵衛餃子・いも天・地酒）
  高知城 → 夜もひろめ市場 or 市内居酒屋 → 高知泊

Day3（5/17 土）
  高知 喫茶店モーニング → 桂浜（坂本龍馬像・太平洋）
  ひろめ市場で最後のカツオたたき
  ドライブ③高知→松山空港（2時間10分）→ 16:30 Jetstar 成田へ
```

## ドライブについて（免許取り立ての方へ）

**高速道路メインで走りやすい構成です。**

| ルート | 道の特徴 | 難易度 |
|--------|---------|--------|
| 道後 → 下灘（県道378号） | 海沿い・夕やけこやけライン・交通量少 | ★★☆☆☆ |
| 下灘 → 内子（国道56号） | 山あいの国道・適度なカーブ | ★★☆☆☆ |
| 道後 → 高知（松山道→高知道） | 高速道路メイン・走りやすい | ★★☆☆☆ |
| 高知 → 松山空港（高知道→松山道） | 高速道路メイン・走りやすい | ★★☆☆☆ |

**レンタカー選び:** コンパクトカー（ヤリス・ノート等）が最初は運転しやすい。  
**注意点:** 妻の体調優先で、1〜1.5時間に一度は必ず休憩を。高速SA（サービスエリア）を積極的に使う。

## ひろめ市場について

[ひろめ市場 公式](https://hirome.co.jp/)｜高知市帯屋町2丁目3-1｜年中無休

約60店舗・430席。席を確保したら各店で注文して持ち寄るスタイル。昼から地酒を傾けるのが高知流。

### 必食リスト

| 店 | メニュー | 特徴 |
|----|---------|------|
| **[明神丸](https://www.myojinmaru.jp/)** | 藁焼きカツオたたき | 目の前で炎が上がるパフォーマンス。塩でどうぞ |
| **安兵衛**（屋台餃子） | 屋台餃子 | 食べログ百名店。タレなしで食べる薄皮餃子 |
| **大平商店** | いも天 | 日曜市の行列店がひろめに出店。甘いさつまいもの天ぷら |
| — | ウツボの唐揚げ | 高知の珍名物。コラーゲン豊富・淡白な味 |
| — | 土佐の地酒 | **酔鯨（すいげい）・司牡丹（つかさぼたん）**など。昼から◎ |

> **作戦:** まず席を確保 → 各店を回って注文 → テーブルで食べ比べ。  
> 近くのテーブルの人と「どこから来た？」と話すのがひろめ流。

> **高知の「おきゃく」文化**  
> 土佐では宴会を「おきゃく」と呼ぶ。ひろめ市場では知らない人とも自然と乾杯になる雰囲気。日本酒の盃を回し合う「献杯・返杯」も土佐の名物。
